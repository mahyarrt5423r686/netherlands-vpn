# اپلیکیشن VPN هلند - بر پایه v2rayNG

این پروژه یک اپلیکیشن اندروید کامل و آماده بر پایه هسته اصلی v2rayNG است. با یک دکمه کاربر به سرور هلند وصل می‌شود و IP او به هلند تغییر می‌کند.

## 🔥 چی ساختم برات؟

1.  **پروژه اندروید کامل** در پوشه `app/` - قابل باز کردن مستقیم در Android Studio
2.  **یک دکمه اتصال** - استایل حرفه‌ای با پرچم هلند
3.  **اتصال واقعی VPN** - از هسته اصلی v2rayNG (libv2ray) استفاده می‌کند، نه فیک
4.  **نمایش IP واقعی** بعد از اتصال
5.  **جای مخصوص برای کانفیگ تو** - فقط لینک خودت رو جایگزین کن

## ⚠️ کانفیگ تو کو؟

تو پیامت گفتی "این کانفیگ رو" ولی هیچ کانفیگی نفرستادی!

باید لینک `vmess://...` یا `vless://...` یا `trojan://...` یا `ss://...` سرور هلندت رو برام بفرستی تا من مستقیم داخل اپ بذارم.

فعلاً من یک جای موقت (Placeholder) گذاشتم که خودت به راحتی می‌تونی جایگزین کنی.

### کجا کانفیگ رو بذارم؟

فایل زیر رو باز کن:
```
app/src/main/java/com/netherlands/vpn/V2RayConfig.kt
```

خط 15:

```kotlin
const val NETHERLANDS_SERVER_URI = "vless://..."
```

لینک خودت رو اینجا بذار. همین!

یا اگر کانفیگ JSON داری، بذار توی:
```
app/src/main/assets/netherlands_config.json
```

## 🚀 چطور APK بسازم؟ (3 دقیقه‌ای)

### روش 1: با Android Studio (پیشنهادی)

1.  Android Studio رو باز کن
2.  File -> Open -> پوشه `NetherlandsVPN` رو انتخاب کن
3.  صبر کن Gradle Sync تمام بشه
4.  اول یک بار باید libv2ray.aar رو دانلود کنی:
    - برو به https://github.com/2dust/AndroidLibXrayLite/releases
    - آخرین نسخه `libv2ray.aar` رو دانلود کن (حدود 20MB)
    - بذار توی `app/libs/` - اگر پوشه libs وجود نداشت بساز
5.  حالا Build -> Build Bundle(s) / APK(s) -> Build APK(s)
6.  APK نهایی توی `app/build/outputs/apk/debug/app-debug.apk` ساخته می‌شه - روی گوشی نصب کن

### روش 2: به صورت forke مستقیم v2rayNG (اگر می‌خوای اپ دقیقا v2rayNG باشه ولی با لوگو و سرور خودت)

من یک اسکریپت هم برات گذاشتم:

```bash
cd NetherlandsVPN
python3 fork_v2rayng.py --config "vless://تو-اینجا"
```

این اسکریپت:
- پروژه اصلی v2rayNG رو کلون می‌کنه
- اسم برنامه رو به "Netherlands VPN" تغییر می‌ده
- آیکون هلند می‌ذاره
- کانفیگ تو رو به عنوان سرور پیش‌فرض اضافه می‌کنه که با اولین اجرا خودکار اضافه بشه
- آماده بیلد می‌کنه

### روش 3: بدون کدنویسی - همین الان تست کن

اگر فقط می‌خوای سریع تست کنی IP عوض می‌شه یا نه:
1. اپ اصلی v2rayNG رو از GitHub دانلود کن
2. کانفیگت رو Copy کن
3. داخل v2rayNG روی + بزن -> Import config from Clipboard
4. اسمش رو بذار Netherlands 🇳🇱
5. وصل شو و به whatismyip.com برو - باید هلند نشون بده

## 🔧 ساختار پروژه

```
NetherlandsVPN/
├── app/
│   ├── build.gradle (وابستگی‌ها + libv2ray)
│   ├── src/main/
│   │   ├── AndroidManifest.xml (مجوز VPN)
│   │   ├── java/com/netherlands/vpn/
│   │   │   ├── MainActivity.kt (UI اصلی + دکمه اتصال)
│   │   │   ├── NLVpnService.kt (سرویس واقعی VPN - هسته v2ray)
│   │   │   └── V2RayConfig.kt (جای کانفیگ هلند)
│   │   ├── res/layout/activity_main.xml (طراحی با پرچم هلند)
│   │   └── assets/netherlands_config.json
│   └── libs/ (اینجا libv2ray.aar رو بذار)
├── fork_v2rayng.py (اسکریپت تبدیل v2rayNG به اپ هلند شما)
├── settings.gradle
└── build.gradle
```

## 🛡️ چطور کار می‌کنه و IP چطور عوض می‌شه؟

1.  اپ از `VpnService` اندروید استفاده می‌کنه (همان API که v2rayNG و همه VPN های واقعی استفاده می‌کنند)
2.  هسته `libv2ray` ( نوشته شده با Go ) یک تونل Socks به پورت 10808 می‌سازه
3.  ترافیک گوشی از طریق TUN به آن Socks می‌ره و از آنجا به سرور هلند شما
4.  سرور هلند ترافیک رو به اینترنت آزاد می‌فرسته -> سایت‌ها IP هلند شما رو می‌بینند

این دقیقا همان کاری است که v2rayNG می‌کند، فقط ما UI را ساده کردیم به "یک دکمه اتصال به هلند".

## ❓ سوالات متداول

**س: آیا بدون libv2ray کار می‌کنه؟**
خیر، هسته لازم است. چون ما نمی‌تونیم اینجا Go کامپایل کنیم، شما یک فایل 20MB باید دانلود کنید (یک بار).

**س: آیا در گوگل پلی می‌تونم منتشر کنم؟**
بله، فقط package name رو در `app/build.gradle` عوض کن (مثلا com.yourname.netherlands) و آیکون و اسم برنامه رو.

**س: می‌خوای من APK رو برات بسازم؟**
کانفیگ VLESS/VMess هلندت رو بفرست، من همینجا داخل کد می‌ذارم و پروژه نهایی رو بهت تحویل می‌دم که فقط Build بگیری.

## 📩 قدم بعدی

کانفیگت رو بفرست (شبیه `vless://uuid@ip:port?type=ws&...#Netherlands`)

من همین الان داخل `V2RayConfig.kt` و `netherlands_config.json` می‌ذارم و یک نسخه نهایی شده بهت می‌دم.

اگر کانفیگ نداری و می‌خوای اول با یک سرور تستی هلند چک کنی، بگو تا یک کانفیگ تست برات بسازم که کارکردش رو ببینی (البته با سرور خودت مطمئن‌تره).
