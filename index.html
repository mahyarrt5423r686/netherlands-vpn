تا من همین الان داخل اپ بذارم و نسخه نهایی را بهت بدهم.

📦 داخل این پروژه چیست؟

  • ✅ پروژه اندروید کامل NetherlandsVPN/app/ قابل باز کردن در Android Studio
  • ✅ سرویس NLVpnService.kt - سرویس VPN واقعی که TUN می‌سازد و ترافیک را به هلند می‌فرستد
  • ✅ MainActivity.kt با طراحی پرچم هلند و نمایش IP واقعی بعد از اتصال
  • ✅ فایل V2RayConfig.kt - فقط کانفیگ خودت را آنجا پیست کن
  • ✅ fork_v2rayng.py - اگر می‌خواهی خود v2rayNG را به نام خودت ریبرند کنی

🚀 چطور در 3 دقیقه APK بسازم؟

قدم 1: Android Studio را باز کن → Open → پوشه NetherlandsVPN را انتخاب کن
قدم 2: هسته v2ray را دانلود کن (فقط یک بار):
برو به AndroidLibXrayLite Releases → فایل libv2ray.aar (حدود 20MB) را دانلود کن و بگذار در app/libs/ . اگر پوشه libs نیست بساز.
چرا این فایل لازم است؟ چون هسته v2ray با Go نوشته شده و باید جداگانه کامپایل شود. پروژه اصلی v2rayNG هم همین کار را می‌کند.
قدم 3: فایل V2RayConfig.kt را باز کن و خط 15 را ویرایش کن:
const val NETHERLANDS_SERVER_URI = "vless://YOUR-UUID@YOUR_NL_IP:443#Netherlands"
لینک واقعی خودت را بگذار.
قدم 4: Build → Build APK → بعد از تمام شدن، فایل در app/build/outputs/apk/debug/ است → روی گوشی نصب کن → دکمه "اتصال به هلند" را بزن → برو whatismyip.com → باید Amsterdam ببینی!

🔧 چطور IP واقعا عوض می‌شود؟ (مراحل فنی)

این اپ دقیقا همان کاری را می‌کند که v2rayNG می‌کند:

  1. از کاربر مجوز BIND_VPN_SERVICE می‌گیرد - یک پنجره سیستمی اندروید
  2. یک TUN interface به آدرس 10.10.10.1 می‌سازد و می‌گوید: همه ترافیک گوشی را به من بده
  3. هسته libv2ray (Go) با کانفیگ هلند شما یک Socks روی 10808 اجرا می‌کند
  4. ترافیک TUN به Socks می‌رود → از آنجا رمزگذاری شده به سرور هلند → سرور به اینترنت آزاد → سایت‌ها IP هلند را می‌بینند
این روش VPN واقعی اندروید است و IP شما 100% عوض می‌شود، نه DNS فیک.

🎨 روش دوم: ریبرند کردن خود v2rayNG

اگر می‌خواهی اپ شما دقیقا ظاهر v2rayNG را داشته باشد ولی با اسم و لوگوی هلند:

python3 fork_v2rayng.py --config "vless://uuid@ip:port#Netherlands"

این اسکریپت پروژه اصلی را کلون، نام را عوض، و کانفیگ هلند را به assets اضافه می‌کند.
