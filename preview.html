🇺🇸 چطور سرور آمریکا اضافه کنی؟

تو خواستی جای آمریکا خالی بمونه که خودت بعدا پر کنی - همین کار رو کردم:

  1. فایل V2RayConfig.kt رو باز کن
  2. خط 20: USA_SERVER_URI
  3. لینک جدیدت رو جایگزین کن:
// قبل (placeholder) const val USA_SERVER_URI = "vless://REPLACE...#USA" // بعد (مثال) const val USA_SERVER_URI = "vless://YOUR_USA_UUID@us.example.com:443?security=tls&type=ws#USA-Server"

یا از داخل خود اپ: روی کارت آمریکا بزن → دیالوگ باز میشه → لینک رو پیست کن → ذخیره موقت → بعدا وارد کد کن برای دائمی.

🚀 چطور APK بسازم؟ (با کانفیگ تو)

  1. Android Studio → Open → پوشه NetherlandsVPN
  2. صبر کن Gradle Sync بشه
  3. فایل libv2ray.aar رو از اینجا دانلود کن (20MB) و بذار توی app/libs/ - اگر پوشه نیست بساز
  4. الان کانفیگ هلند تو (mahyar1) خودکار داخل اپه - نیازی به ویرایش نیست! ولی اگر آمریکا رو هم میخوای همین الان اضافه کن
  5. Build → Build APK(s) → APK توی app/build/outputs/apk/debug/
  6. روی گوشی نصب کن → یک سرور انتخاب کن → اتصال → چک IP
نکته: بدون libv2ray.aar اپ کامپایل میشه ولی VPN واقعی وصل نمیشه (نسخه دمو). با اضافه کردنش، هسته اصلی Xray اجرا میشه و IP واقعا عوض میشه.
